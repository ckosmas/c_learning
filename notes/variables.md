------------
Variables
------------
Reusable containers for a value and behaves aas of it were the value it contains.
int = whole numbers which are 4 bytes in modern systems (%d)
float = single precision demical number, 4 bytes (%f)
double = double precision decimal number, 8 bytes (%lf)
char = single character, 1 byte (%c)
char [] = array of characters, size varies (%s)
bool = true or false, 1 byte and requires <stdbool.h>
